# OTF3K 🍀 — One Dream. One World.

Single-file site: every image is embedded inside `index.html`. No assets folder needed — nothing can go missing.

## Publish on GitHub Pages
1. Create a **public** repo at github.com/new
2. Upload `index.html` and `.nojekyll`, commit
3. Settings → Pages → Deploy from a branch → **main** → **/ (root)**
4. Live in ~1 minute at `https://YOUR-USERNAME.github.io/REPO-NAME/`

To update later, just replace `index.html`.
